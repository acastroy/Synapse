play_vlc::SYNAPSE_REGISTER ( play_Open  )

Opens and starts playing an url.

  • url The url or local filename to open.
Broadcasts: play_InfoMeta:
Metadata for the current url. (can also be sended while playing, for streams for example)
Broadcasts: play_StateNone:
Player has no status.
Broadcasts: play_StateOpening:
Player is opening an url
Broadcasts: play_StateBuffering:
Player is buffering data
Broadcasts: play_StatePlaying:
Player is playing.
Broadcasts: play_StatePaused:
Player is paused
Broadcasts: play_StateStopped:
Player has stopped
Broadcasts: play_StateEnded:
Media has ended
Broadcasts: play_StateError:
An error has occured. TOOD: need to fix logging, to show what actually went wrong.
Broadcasts: play_Time:
Sended every second with time and position info.
  • length total length of media in seconds. (0 for streams)
  • time current position of the media, in seconds.
